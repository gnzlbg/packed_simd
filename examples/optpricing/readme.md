# Options Pricing ISPC example

This is the [`options` ISPC benchmark][ispc]:

> This program implements both the Black-Scholes and 
> Binomial options pricing models.

## Usage

```
cargo run --release --features=ispc -- ${SIZE} ${ALGORITHM}
```
